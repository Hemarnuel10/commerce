# CS50 Commerce – Online Auction Platform

## Overview

CS50 Commerce is an online auction website built with the Django framework as part of Harvard's CS50 Web Programming course.

The application allows users to create auction listings, place bids, comment on listings, and manage personal watchlists. It demonstrates how to build a full-stack web application using Django, including models, authentication, forms, and database interactions.

---

## Features

### User Authentication
- Register new users
- Login and logout functionality
- Secure authentication system

### Auction Listings
- Create new auction listings
- View all active listings
- Close auctions
- Display winning bidder

### Bidding System
- Users can place bids
- System validates bids to ensure they are higher than the current price
- Displays the current highest bid

### Watchlist
- Users can add listings to their personal watchlist
- Remove items from the watchlist
- View all saved listings

### Comments
- Users can leave comments on listings
- Comments are displayed under each listing

### Categories
- Browse listings by category
